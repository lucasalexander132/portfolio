- Create `lib/utils.ts` with cn() helper:
  ```typescript
  import { type ClassValue, clsx } from "clsx"
  import { twMerge } from "tailwind-merge"

  export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs))
  }
  ```

Note: Next.js 16 uses Turbopack by default, no additional config needed. Tailwind CSS 4 is included via create-next-app.

  - Fluid type scale using clamp() (--text-display through --text-small)
  - Color palette: base-950 through base-600 (deep slate), amber-600 through amber-300 (golden honey), text colors (warm off-white)

  - p: body font size, max-width 65ch
